# @saber2pr/json-type

> convert json to d.ts declaration.

```bash
# from npm
npm install @saber2pr/json-type -g

# from github
git clone https://github.com/Saber2pr/json-type.git
```

# Usage

```bash
jsonType ./test.json

jsonType ./test/test.json ./outDir

```

---

## start

```bash
npm install
```

```bash
npm start

npm test

```
